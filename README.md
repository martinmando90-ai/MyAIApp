# 🤖 My AI Assistant - مساعدي الذكي

## 📱 تطبيق AI يشتغل على الموبايل

### المميزات:
- ✅ دردشة ذكية (Online & Offline)
- ✅ بحث على الإنترنت
- ✅ قراءة ملفات (PDF, Word, TXT)
- ✅ تذكيرات ومنبهات
- ✅ صوتي (تحدث بدل الكتابة)
- ✅ أكتر من لغة
- ✅ Dark/Light Mode

### 🚀 خطوات التشغيل:

1. **افتح Android Studio**
2. **File → Open** واختار مجلد MyAIApp
3. **انتظر Gradle** يبني المشروع
4. **اضغط Run** (Shift+F10)
5. **اختار جهازك** أو Emulator

### 🔑 إضافة API Keys:

افتح `local.properties` وأضف:
```
GEMINI_API_KEY=your_key_here
```

### 📞 للدعم:
- Android Docs: developer.android.com
- Gemini API: ai.google.dev
